## associative, commutative, and distributive properties

### associative

a(b*c) = (a*b)*c
2(3*4) = (2*3)*4

a(bc) = (ab)c
2(12) = 16*4

abc = abc

### commutative rule

a*b = b*a
a*b*c = b*a*c = c*a*b 

### distributive rule

x(y+z) = x*y + x*z
